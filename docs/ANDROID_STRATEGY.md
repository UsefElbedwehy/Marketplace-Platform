# Android Strategy

> Not yet started. Planned in Phase 8 — see [planning/11-roadmap.md#phase-8](planning/11-roadmap.md#phase-8--hardening-scale--next-platforms).

Android (and Web) consume the same versioned REST contract and the same category/attribute/theme metadata as iOS — this is mostly a new client build, not new backend work. Detailed strategy will be written once the contract and dynamic schema engine are proven end-to-end on iOS (Phase 5 milestone).
